## 💫 About Me

I'm a full-stack engineer, frontend-leaning by background, who builds products end-to-end — from the database and API to the UI, all in TypeScript. Lately I spend a lot of that time on AI-native features: agent loops, tool-calling, and LLM orchestration. I like to understand these things from first principles rather than just wiring up a library, and I approach my work with a product mindset — putting on that hat and building things that solve real problems.

## 🚀 Currently

- 🧩 **Senior Frontend Developer @ Sanstech** — building and maintaining large-scale web applications, and contributing to the company's internal AI transformation by weaving AI into everyday workflows.
- 🧾 **Co-founder @ [Fişle](https://fisle.co)** — an AI-powered accounting SaaS. I own the software side end-to-end: extraction pipelines, the Next.js app, and the Cloudflare/Workers backend.

## ⭐ Highlights

- 🧠 **[porti-code](https://github.com/fport/porti-code)** — an open-source coding agent built from scratch (Claude Code-style). I built it to understand agentic systems from first principles: the agent loop, tool-calling, and LLM orchestration — not just consuming an SDK.
- 🏗️ Designed and built an end-to-end internal product — owning the full stack across frontend, backend, and DevOps, with a particular focus on the AI layer.
- ⚙️ Led a large-scale JavaScript → TypeScript migration and a scss|styled → Tailwind CSS transition, improving maintainability, performance, and scalability.
- 🤖 Shipped AI features in production — multi-provider LLM pipelines, MCP tool-use, and retrieval (embeddings, pgvector).
- 🎨 Full-stack in TypeScript, end-to-end — semantic HTML and accessibility on the frontend, clean API and data design on the backend.
